# Customer Shopping Behavior Analysis

This project analyzes customer shopping patterns using Python, SQL, and Power BI.

Key Analysis:
- Customer purchasing trends
- Popular product categories
- Spending patterns by demographics

Tools Used:
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI

Dataset:
Customer shopping behavior dataset.

Note:
This project was inspired by an open-source analysis project and was uploaded for learning and portfolio purposes.
