# Customer Shopping Behavior Analysis

## Overview
This project analyzes 3,900 customer purchase records to identify shopping patterns, product preferences, and spending behavior. The analysis combines Python, SQL, and Power BI to generate insights that can support business decision-making and marketing strategies.

## Dataset
The dataset contains 18 columns including:

- Customer demographics (Age, Gender, Location)
- Purchase details (Item Purchased, Category, Purchase Amount)
- Shopping behavior (Discount Applied, Frequency of Purchases)
- Customer feedback (Review Rating)
- Subscription status and shipping type

## Analysis Workflow

### 1. Data Preparation (Python)
- Data loading and exploration
- Handling missing values
- Data cleaning and column standardization
- Feature creation (e.g., age groups)

### 2. Business Analysis (SQL)
Key questions explored:
- Revenue comparison by gender
- Spending patterns of subscribers vs non-subscribers
- Identification of high-spending customers using discounts
- Top-rated products based on customer reviews
- Revenue contribution by different age groups

### 3. Data Visualization (Power BI)
An interactive Power BI dashboard was created to visualize:
- Revenue trends
- Product category performance
- Customer segmentation insights
- Purchasing patterns
